# Coffee_shop
A modern Coffee Shop website built using HTML, CSS, and JavaScript. The project features a responsive design, interactive menu, shopping cart functionality, reservation form, testimonials section, and a clean user-friendly UI inspired by modern café websites.
## Features
- Modern Responsive Navbar
- Home, About, Menu, and Contact Sections
- Smooth Scrolling Navigation
- Interactive Shopping Cart
- Reservation & Contact Form
- Customer Testimonials Section
- Clean and User-Friendly UI
- Fully Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScrip
